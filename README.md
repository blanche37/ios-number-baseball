### 🍅 프로젝트 🍅
- 브랜치 이름을 정할때 , 닉네임-기능으로 만들기. ex) yun-feature-1, sunny-feature-2, coda- feature-3
- 커밋 메시지 제목 + 1~2 문장 이내로 설명문 작성하기
- 커밋 메시지 제목에 각자의 작업 순서 번호 추가하기. ex) [#1] yun, [#2] sunny, [#3] coda
